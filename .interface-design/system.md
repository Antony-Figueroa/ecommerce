# Sistema de Diseño de Ana's Supplements

## Dominio: Rendimiento y Vitalidad
- **Conceptos**: Salud, Frescura, Energía, Pureza, Recuperación.
- **Vocabulario**: Vital, Sereno, Puro, Zen, Flujo.

## Mundo de Color: Vitalidad Manzana
- **Primario**: #8fca1d (Verde Manzana / Vitalidad Fresca)
- **Secundario**: #f3f0e8 (Hueso / Calidez Humana)
- **Acento**: #e5c9a0 (Arena / Energía Suave)
- **Fondo**: #fbfaf8 (Crema muy claro / Pureza)
- **Texto**: #2c3327 (Gris profundo / Lectura Suave)
- **Superficies**: Blanco con bordes sutiles en #e5e0d5.

## Firma: El "Toque de Frescura"
Un diseño vibrante y amigable con esquinas muy redondeadas (16px), generoso espacio en blanco y acentos verde manzana que evocan una sensación de energía positiva, salud y bienestar moderno.

## Estrategia de Profundidad: Suavidad Sutil
- Usamos bordes ligeros para la estructura.
- Los estados de interacción (hover) usan un suave resplandor olivo o un ligero levantamiento crema.

## Tipografía: Profesional y Cercana
- **Titulares**: Inter Negrita (Espaciado equilibrado)
- **Datos**: Sans-serif clara para números.
- **Cuerpo**: Inter Regular para máxima legibilidad.

## Espaciado: Base de 8px (Estandarizado)
Todos los márgenes y rellenos siguen una cuadrícula de 8px para una sensación más relajada y profesional.

## Estándares a Seguir:
1. **Tarjeta Genérica** -> **Tarjeta Vital**: Acentos olivo suaves + radio de 16px + superficie blanca.
2. **Barra Lateral Estándar** -> **Navegación Limpia**: Etiquetas de alto contraste + indicadores olivo.
3. **Caja de Métricas** -> **Estadística de Salud**: Iconos olivo + números pizarra.

## Panel Admin: Dirección Operativa
- **Intención**: Sereno, clínico y operativo. La información crítica aparece primero.
- **Jerarquía**: Resumen arriba + acciones rápidas + lista operacional.
- **Métricas clave**: Ventas totales, pedidos pendientes, productos críticos, ticket promedio.
- **Inventario**: Cobertura promedio, movimientos recientes, stock bajo/crítico/sobrestock.
- **Tablas**: Encabezados en muted, separadores suaves, hover en muted/30.
- **Filtros**: Chips compactos con estados activos y contadores discretos.
