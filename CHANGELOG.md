# Changelog - Ana's Supplements

Todas las actualizaciones notables de este proyecto serán documentadas en este archivo.

## [1.1.0] - 2026-02-08

### ✨ Nuevas Funcionalidades
- **Mejoras en UI y Analíticas**: Se implementó un tema oscuro más refinado, mejoras en la carga de la interfaz y un panel de analíticas para administradores más completo.
- **Sistema de Animaciones**: Integración de `framer-motion` para transiciones más fluidas y un sistema de diseño UI mejorado.
- **Gestión de Ventas y Propuestas**: Se añadió un sistema de propuestas para modificaciones de órdenes en el módulo de ventas.
- **Perfiles de Usuario**: Ahora se permite gestionar direcciones de usuario y se han mejorado las capacidades de administración de perfiles.
- **Documentos y Reportes**: Generación de catálogos en PDF, facturación integrada y soporte para múltiples categorías por producto.

### 🔧 Mejoras
- **Refactorización de Admin**: Limpieza de importaciones y optimización del diseño de la barra lateral (sidebar).
- **Rendimiento React**: Optimización mediante el uso de operaciones asíncronas en paralelo y memoización para reducir re-renders.
- **Dashboard y Notificaciones**: Optimización general del panel de control, sistema de notificaciones y herramientas analíticas.
- **Arquitectura del Servidor**: Reestructuración del backend para una mejor escalabilidad y limpieza del storefront.

### 🐛 Correcciones
- **Tipos de Prisma**: Corrección de inconsistencias en los tipos generados por Prisma.
- **Seguridad**: Ajuste de políticas COOP para el sistema de autenticación.
- **UI de Productos**: Correcciones visuales en la visualización de productos, gestión de roles y panel de órdenes.

### 📝 Documentación
- **Guías para Agentes AI**: Adición de directrices obligatorias para el uso de skills por parte de agentes de IA.
- **Documentación Técnica**: Mejora y actualización de la documentación del proyecto, incluyendo arquitectura y descripción general.

## [1.0.0] - 2026-02-05

### 🚀 Lanzamiento Inicial
- Respaldo inicial del proyecto y configuración base del ecosistema Ana's Supplements.
- Configuración de Frontend (Vite + React) y Backend (Express + Prisma).
